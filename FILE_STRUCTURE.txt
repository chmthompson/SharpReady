Key File Descriptions
=====================

Core Application Files
- MauiProgram.cs: Configures dependency injection container. All services and ViewModels are registered here.
- App.xaml/cs: Application-level resources and lifecycle management.
- AppShell.xaml: Defines navigation structure with tabs for Main and Profile.

UI/Views
- Views/MainPage.xaml: Counter demo with data binding examples
- Views/ProfilePage.xaml: Profile page template
- AppShell.xaml: Tab-based navigation with two pages

Business Logic
- ViewModels/MainViewModel.cs: Implements MVVM pattern with properties and commands
- Services/IDataService.cs: Abstract interface for data operations
- Services/MockDataService.cs: In-memory implementation for development

Data Models
- Models/BaseModel.cs: Base class providing Id, CreatedAt, UpdatedAt, Notes
- Models/UserProfile.cs: Example entity (FirstName, LastName, Email, Phone, Bio)

Infrastructure
- Utilities/BaseViewModel.cs: Provides INotifyPropertyChanged for data binding
- Services/NavigationService.cs: Wrapper around MAUI Shell routing

Android-Specific
- Platforms/Android/MainActivity.cs: Android activity configuration
- Platforms/Android/MainApplication.cs: Android application lifecycle
- Platforms/Android/AndroidManifest.xml: Permissions and app configuration

Getting Started Commands
========================

Navigate to project:
  cd D:\Code\claude\SharpReady

Build:
  dotnet build -f net9.0-android

Run on emulator:
  dotnet run -f net9.0-android

  dotnet run -f net9.0-android

Clean:
  dotnet clean

Restore:
  dotnet restore
